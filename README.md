# Ingeniería de Software: Ciclos de Vida y Agilidad

Este es un recurso educativo interactivo diseñado para estudiantes de ingeniería de software. Cubre los conceptos fundamentales del Ciclo de Vida de Desarrollo de Software (SDLC) y las Metodologías Ágiles (Scrum).

## 🚀 Características
- **Guía Teórica**: Explicaciones claras sobre modelos tradicionales (Cascada, Espiral) y Ágiles.
- **SDLC Explorer**: Simulador interactivo para visualizar el flujo de un requerimiento en un Sprint.
- **Actividad Diagnóstica**: Caso práctico con retroalimentación inmediata.
- **Diseño Premium**: Interfaz moderna con efectos de glassmorphism y micro-animaciones.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Assets visuales generados con IA.

## 📦 Publicación en GitHub Pages
Para hacer este recurso público:
1. Sube estos archivos a un repositorio de GitHub.
2. Ve a **Settings > Pages**.
3. En **Build and deployment**, selecciona la rama `main` y la carpeta `/ (root)`.
4. ¡Listo! Tu recurso será accesible mediante una URL pública.

## 📁 Almacenamiento en Google Drive
Este recurso está diseñado para ser ligero. Los estudiantes pueden:
- Usar el botón de **"Descargar Bitácora de Progreso"** (implementado en la versión interactiva) para guardar sus reflexiones en formato `.md` o `.txt` y subirlas a su carpeta compartida de Google Drive.
